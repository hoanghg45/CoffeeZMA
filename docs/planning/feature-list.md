### 2. Menu

**What it does:** Your products. All of them. Easy to change.

**Core:**
- Add/edit/delete products
- Upload image, set price, write description
- Manage categories (Coffee, Tea, Food)
- Manage options (Size, Sugar, Ice, Toppings)
- Toggle availability (In Stock / Out)

**Why:** The menu changes. This makes it effortless.

### 3. Customers

**What it does:** Know who buys. What they buy. How often.

**Core:**
- Customer list with purchase history
- Loyalty points balance
- Segment: New / Regular / VIP
- Search by name or phone

**Why:** Customers return when you know them.

---

### 4. Promotions

**What it does:** Create discounts. Track results.

**Core:**
- Voucher codes (% or fixed amount)
- Set rules: minimum order, usage limit, expiry
- Track redemptions
- On/off toggle

**Why:** Smart promotions drive sales. This keeps it simple.

---

### 5. Analytics

**What it does:** Revenue. Products. Trends.

**Core:**
- Revenue by day/week/month
- Best-sellers list
- Peak hours chart
- Export to Excel

**Why:** Data tells truth. This shows it clearly.

---

### 6. Team

**What it does:** Add staff. Control access.

**Core:**
- Create accounts for staff
- Roles: Admin / Manager / Staff
- Permission levels per role
- Activity log

**Why:** Trust your team. Verify with permissions.

---

### 7. Settings

**What it does:** Store config. Delivery fees. Global toggles.

**Core:**
- Store name, hours, contact
- Delivery fee per km
- Payment methods
- System preferences

**Why:** One place for everything that doesn't change often.
